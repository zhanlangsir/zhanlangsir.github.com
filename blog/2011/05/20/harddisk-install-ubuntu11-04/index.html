硬盘安装ubuntu11.04

May 20th, 2011

今天想升级到11.04，刻盘后发现刻坏了，无法启动，真悲剧，只能硬盘安装了~

1.下载软件 操作系统ubuntu：http://www.ubuntu.com/desktop/get-ubuntu/download 系统引导软件EasyBCD：够快下载

[more] 2.设置NeoGrub启动项 1）把ubuntu的iso文件放到C盘根目录，并将iso中的casper目录下的vmlinuz和 initrd.lz解压到C盘根目录下 2）安装EasyBCD 3）启动EasyBCD 4）选择 Add New Entry -> NeoGrub -> Install -> Configure -> 将如下代码粘贴到自动打开的记事本中 title Install Ubuntu 10.10 root (hd0,0) kernel (hd0,0)/vmlinuz boot=casper iso-scan/filename=/ubuntu-11.04-desktop-i386.iso ro quiet splash locale=zh_CN.UTF-8 initrd (hd0,0)/initrd.lz 注：记事本代码中的ubuntu-11.04-desktop-i386.iso要和下载的ubuntu文件名字一致

3.硬盘安装ubuntu 1）重启电脑选择NeoGrub Bootloader启动项进入ubuntu live cd桌面 2）打开终端，或者alt+F2,输入如下命令 sudo umount -l /isodevice回车执行 3）点击桌面上的安装ubuntu一步步按照提示选择安装

4.调整默认启动项为WIN7 1）终端执行： sudo gedit /boot/grub/grub.cfg 2）找到default 0把0改为4就可以了（4为WIN7的启动项顺序号） 6.删除NeoGrub启动项 1）进入WIN7桌面启动EasyBCD，选择 Add New Entry -> NeoGrub -> remove 删除NeoGrub启动项 2）删除C盘根目录下的vmlinuz和 initrd.lz文件

那么ubuntu的硬盘安装就完成了~是不是很简单呢~
