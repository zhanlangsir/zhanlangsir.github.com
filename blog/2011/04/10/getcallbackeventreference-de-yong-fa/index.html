GetCallbackEventReference的用法

Apr 10th, 2011

作用：获取一个对客户端函数的引用；调用该函数时，将启动一个对服务器端事件的客户端回调。此重载方法的客户端函数包含指定的控件、参数、客户端脚本和上下文。 函数原型： public string GetCallbackEventReference ( Control control, string argument, string clientCallback, string context )
