Ubuntu10.10 SCIM输入法一些问题的解决办法

Apr 10th, 2011

 

1、不能光标跟随

2、输入几个字后按BackSpace键，输入法不停地闪烁，无法继续输入

sudo apt-get install scim scim-pinyin scim-tables-zh im-switch scim-qtimm scim-bridge scim-bridge-client-gtk scim-bridge-client-qt scim-bridge-agent # im-switch -s scim -z default

编辑im-switch生成的scim配置文件

sudo gvim /etc/X11/xinit/xinput.d/scim

将默认的 GTK_IM_MODULE=xim QT_IM_MODULE=xim 修改为 GTK_IM_MODULE=scim QT_IM_MODULE=scim 保存退出,注销一下。看看是不是已经解决了，不过这时候，可能会出现两个托盘图标，可以把其中一个的隐藏掉，在图标上右键，进入SCIM设置，面板－》GTK，把显示托盘图标选项勾掉就可以了。
