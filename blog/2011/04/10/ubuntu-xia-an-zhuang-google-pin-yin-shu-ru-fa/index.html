Ubuntu下安装Google拼音输入法

Apr 10th, 2011

在Ubuntu下安装Google拼音输入法，如下： A. 获取代码：（没有git的先安装git：sudo apt-get install git-core） $ git clone git://github.com/tchaikov/scim-googlepinyin.git $ cd scim-googlepinyin B. 编译前提： 上面给的链接里面有介绍怎么编译的，但少提了几个必需组件，这里列一下： * autotools-dev * libgtk2.0-dev * libscim-dev * libtool * automake 用下面命令看看是不是安装了，如果没有，会自动帮你安装上： $ aptitude install autotools-dev libgtk2.0-dev libscim-dev libtool automake C. 编译： 记住系统必须先存在SCIM（没有的话 sudo apt-get install scim 一下） $ ./autogen.sh $ make $ sudo make install 现在重启scim: 关闭scim:pkill scim 然后启动： scim -d 去首选项里的“语言支持”的“输入法”选择scim作为默认输入法,然后注销就OK了。
