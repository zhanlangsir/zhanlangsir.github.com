Windows下安装pyclewn

Apr 10th, 2011

  昨天在windows下安装pyclewn时，老是提示”netbeans support vim is require“错误，我也查看了vim的version，没错啊。。已经开了这个特性了。郁闷，因为没学过python,百度,google也找不到类似的错误，更加没有高手相助，所以只能自己多方尝试了，我就自己重新编译vim,然后又找补丁，看有没有相关的说明，稍微有的沾边的都给vim打上，然后再编译，查看手册。。。。。等等，最后发现竟然是因为python的版本不对，python3语法有所改变，必须用python2(pyclewn目前最新版本1.4),而我下的是python3…赶紧去下了一个2.7.1的版本，安装完后发现还是有错误，说找不到某个模块导入，然后再下了个相应版本的Python for Windows extensions，记得下exe格式的安装版本，自己编译的话又会有一堆别的错误了。。。。。。忙活了大白天，真实内牛满面啊。。 现在附上上面提及的东西的下载地址： pyclewn:http://sourceforge.net/projects/pyclewn/files/ python2 、 3：http://python.org/ Python for Windows extensions：http://sourceforge.net/projects/pywin32/
