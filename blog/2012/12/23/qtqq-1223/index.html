Qtqq 12/23

Dec 23rd, 2012

12/23更新:

版本:v0.8.0-alpha

新特性:

1.聊天窗口使用tab页

其他修改：

1.关闭气泡消息提示

2.新消息提醒改为托盘图标菜单

3.搜索结果显示方式改变

关于qtqq v0.8.0-alpha功能的说明可以看这里:

关于qtqq0-8-0-alpha

安装方法:
ppa没人维护了，有熟悉的童鞋可以接手

ppa: sudo add-apt-repository ppa:qtqq/qtqq-stable sudo apt-get update

sudo apt-get install qtqq

 

手动编译:
git clone https://github.com/zhanlangsir/Qtqq.git qtqq
cd qtqq
git checkout qtqq-dev
mkdir release
cd release
cmake -DCMAKE_BUILD_TYPE=Release  -DCMAKE_INSTALL_PREFIX=/usr  .. (后面有两点,表示的是qtqq代码的根目录)
make
sudo make install

 

fedora需要准备的环境:

需要安装git cmake qt-devel  gcc gcc-c++ qtwebkit-devel, yum install即可.

 

ubuntu需要准备的环境:

需要安装git cmake g++ qt4-dev-tools
另外fedora64位需要把/usr/lib下的so文件手动移动到/usr/lib64
还有其他安装事项请查看根目录下的INSTALL文件或者https://github.com/zhanlangsir/Qtqq/wiki
