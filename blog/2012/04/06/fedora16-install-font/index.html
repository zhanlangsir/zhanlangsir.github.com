Fedora16安装新字体

Apr 6th, 2012

1.将windows字体拷贝到/usr/share/fonts/truetype下面，文件夹名字可以随便起 cp /media/c/WINDOWS/Fonts/* /usr/share/fonts/truetype/ 默认字体权限应该是755，如果不是的话，手动更改下 chmod 755 * 2.在/usr/share/fonts/truetype下面运行 mkfontscale mkfontdir 3.运行fc-cache -fv，刷新字体缓存。 现在就一切ok了

当然，前提是先su获取root权限～
